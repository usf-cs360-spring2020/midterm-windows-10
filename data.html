Data Source

This project is supposed to visualize certain relationships between airline data from the San Francisco International Airport

Data acquired from DataSF on February 9, 2020.

  1. Activity Period: The year and month at which passenger activity took place. It's a numeric type with a range from 200507-201909.
  2. Operating Airline: Airline name for the operator of aircraft with passenger activity. It is a text type, with a range of Airline names.
  3. Operating Airline IATA Code:The International Air Transport Association (IATA) two-letter designation for the Operating Airline. It is a text type.
  4. Published Airline: Airline name that issues the ticket and books revenue for passenger activity. It is a text type
  5. Published Airline IATA Code: The International Air Transport association (IATA) two-letter designation for the Published Airline. It is a text type
  6. GEO Summary: Provides a more detailed breakdown of the GEO Summary field to designate the region in the world where activity in relation to SFO arrived from or departed to without stops. It is a text type.
  7. GEO Region:Provides a more detailed breakdown of the GEO Summary field to designate the region in the world where activity in relation to SFO arrived from or departed to without stops. It is a text type.
  8. Activity Type Code: A description of the physical actiona passenger took in relation to a flight, which includes boarding a flight (“enplanements”), getting off a flight (“deplanements”) and transiting to another location (“intransit”). It is a text type.
  9. Price Category Code: A categorization of whether a Published Airline is a low-cost carrier or not a low-cost carrier. It is a text type.
  10. Terminal: The airport terminal designations at SFO where passenger activity took place.It is a text type.
  11. Boarding Area: The airport boarding area designations at SFO where passenger activity took place. It is a text type
  12. Passenger Count: The number of monthly passengers associated with the above attribute fields.It is a numeric field.

For these specific visualizations, I ended up using

  1. Passenger Count
  2. Activity Period (Converted to Year)
  3. GEO Country (Registered Country)
  4. Boarding Area

Wrangling

I originally downloaded the whole dataset since I wanted to work with more categories.

The data had to be sorted and fitlered out and I ended up doing it based on the time periods, I filtereted out uncommon and unecessary data manually through tableu functions prior to importing into javscript.

However, I did use the javascript sort function in order to reorder the data based on ascending and/or alphabetical order

It was important to see the relevance and the importance of the data.
